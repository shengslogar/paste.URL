paste.URL
======

Summary
-------
Simple Chrome extension to paste and navigate to a copied URL.

[Find it on the Chrome Web Store.](https://chrome.google.com/webstore/detail/pasteurl/pdmdajhcdelodpkghpidenngfienklem)
